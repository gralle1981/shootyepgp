Custom ranks included in this build:

Top MS priority:
- Guild Master
- Guild Council
- Loot Master
- Officer
- Raider

Then:
- Firkløver(MS)
- Guild Master/ Guild Council/ Loot Master/ Officer/ Raider (OS)
- Firkløver(OS)
- Officer Alt(MS), Alt(MS)
- Officer Alt(OS), Alt(OS)

This means alts are below OS even if they roll MS.

Officer note handling:
The addon only adds/replaces a [Rank] tag and leaves the rest of the officer note intact.
Example: {3427:582} -> [Raider]{3427:582}
